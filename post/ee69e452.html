font字体
body {
    font-size: 16px;
}

  1. font-size: 大小

    • em: 相对于当前对象内文本的字体尺寸
    • px: 像素
    • in：英寸
    • cm：厘米
    • mm：毫米
    • pt：点

  2. font-family: 字体

    多字体使用逗号+空格隔开，都没有则以电脑默认字体为准

    字体内包含空格、#、$ 等符号，需使用引号（单双都可）

    Unicode字体：

    字体名称 英文名称 Unicode 编码
    宋体 SimSun \5B8B\4F53
    新宋体 NSimSun \65B0\5B8B\4F53
    黑体 SimHei \9ED1\4F53
    微软雅黑 Microsoft YaHei \5FAE\8F6F\96C5\9ED1
    楷体_GB2312 KaiTi_GB2312 \6977\4F53_GB2312
    隶书 LiSu \96B6\4E66
    幼园 YouYuan \5E7C\5706
    华文细黑 STXihei \534E\6587\7EC6\9ED1
    细明体 MingLiU \7EC6\660E\4F53
    新细明体 PMingLiU \65B0\7EC6\660E\4F53

  3. font-weight: 字体粗细

    也可用标签实现（b, strong)

    font-weight: normal | bold（加粗） | number
number: 100-900 ,其中700 = bold, 400 = normal

  4. font-style: 字体风格

    也可用标签实现(i, em)

    font-style: normal | italic(斜体)

  5. 综合

    选择器{
    font: font-style font-weight font-size/line-height font-family;
}
<!-- 其中size和family不可缺省 -->

标签显示模式

​ HTML标签分为块标签和行内标签两种类型，它们也称为块元素和行内元素

  1. ​ 块级元素
    常见的块元素有<h1>~<h6>、<p>、<div>、<ul>、<ol>、<li>等，其中<div>标签是最典型的块元素
    块级元素特点：

    (1). 自己占一行

    (2). 宽度，高度，外边距以及内边距都可控制

    (3). 宽度默认为容器的100%

    (4). 是一个容器及盒子，里面可以放行内或者块级元素

    p内不可放块级元素，因为只有文字才能组成段落

    同理h1, h2, …, h6, dt这些

  2. 行内元素
    常见的行内元素有<a>、<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<span>等，其中<span>标签是最典型的块元素，有的地方也称内联元素

    行内元素特点：

    (1). 相邻行内元素在一行上，一行可以显示多个

    (2). 宽度，高度，设置无效

    (3). 宽度默认为本身内容的宽度

    (4). 行内元素只能容纳文本或其他行内元素

    链接内不可再放链接

    特殊情况a内可放块级元素，但是给a转换一下块级模式比较安全

  3. 行内块元素
    、、，可以对它们设置宽高和对齐属性
    行内块元素的特点：

    (1). 和相邻行内元素（行内块）在一行上，但是之间会有空白缝隙，一行可显示多个
    (2). 默认宽度为它本身内容的宽度

    (3). 高度，行高，外边距以及内边距都可以控制

  4. 标签显示模式转换

    display: inline;	/*块转行内*/
display: block;		/*行内转块*/
display: inline-block;	/*块，行内元素转行内块*/
line-height（行高）

  1. 单行文本水平居中

    <style>
    div {
        height: 100px;
        width: 50px; 
        <!-- 行高等于宽度 -->
        line-height: 50px; 
    }
</style>
<div>
    行内文字
</div>
    • 行高等于高度，文字垂直居中
    • 行高大于宽度，文字偏下
    • 行高小于宽度，文字偏上
CSS背景

  1. 背景颜色（color）

    background-color: transparent | none   //默认值为transparent

  2. 背景图片（image）

    background-image: none | url() //地址无需加引号
    • 默认平铺整个盒子，若图片太小会重复出现直至铺满盒子

  3. 解决图片重复

    background-repeat: no-repeat | repeat-x | repeat-y | repeat

  4. 背景位置（position）

    background-position: length || length
background-position: position || position
    • length: 百分数 / 由浮点数或者单位标识符组成的长度值
    • position： top | center | bottom | left | right 方位名词
    • background-position: 后跟x、y坐标

  5. 背景附着

    background-attachment: scroll | fixed
    • scroll ： 背景图像随对象内容滚动
    • fixed ： 背景图像固定

  6. 背景简写

    background: color  image  repeat  attachment  position

  7. 背景透明

    background-color: rgba(0, 0, 0, 0~1) /*第四个参数为不透明度，*/
CSS特性

  1. 层叠性

    • 对相同标签进行重复处理，优先处理后面的

  2. 继承性

    • 子标签会继承父标签的某些样式（text-， font-， line-，color ）
    1. 优先级

    • 定义多种规则在同一元素，根据优先级判断应用，优先级相同执行层叠性

      权值表：

      标签选择器 权值：1
      伪元素选择器或伪对象选择器 权值：1
      类选择器 权值：10
      属性选择器 权值：10
      id选择器 权值：100
      继承或* 权值：0

      行内样式 > 局内样式 > 外部样式 > 导入样式

      style = “” 权值：1000

      ！important 权值：无穷大

    • 权重叠加

      .class a {
    
}

      采取简单加法
      div ul li –> 0 ,0, 0, 3

      .nav ul li –> 0, 0, 1, 2

      a: hover –> 0, 0, 1, 1

      .nav a –> 0, 0, 1, 1

      叠加无进位，继承的权重为0

盒子模型：

  1. 盒子边框

    border:border-width || border-style || border-color
border-width: 边框粗细，单位为px
border-style: 边框样式
border-color: 边框颜色

    • 边框的样式：
      none: 没有边框

      solid: 单实线
      dashed：边框为虚线

      dotted: 边框为点线
      ……

    • 简写

      border: width style color;

    • 对四个边框单独指定

      border-top:
border-bottom:
border-left:
border-right:

    • 单元格边框过粗（由于叠加）使用border-collapse

      border-collapse: collapse /*合并相邻的边框*/

  2. 内边距

    padding-left: 10px;
padding-right:
padding-top:
padding-bottom:

/*简写*/
padding: xx    /*上下左右*/
padding: 10px 20px	/*上下,左右*/
padding: 10px 20px 30px	/*上,左右,下*/
padding: 10px 10px 10px 10px	/*上右下左*/

    添加内边距后，盒子会变大

    盒子的实际大小 = 内容宽度 / 高度 + 内边距 + 边框

    解决方法，减去内边距，即减小width

特殊：不指定宽度 / 高度，padding不会撑开盒子

  1. 外边距(与padding基本相同)

    margin-left: 10px;
...
margin: xx
...

  2. 块级盒子水平居中

    ​ 条件：盒子必须指定宽度

    ​ 左右外边距设置为auto

    1. margin-left: auto; margin-right: auto;
2. margin: auto;
3. margin: 0 auto;

  3. 文字居中和盒子居中区别

    文字居中： text-align: center /*同时可使盒子里的文字、行内元素以及行内块元素居中*/
盒子居中： 如上

  4. 插入图片与背景图片区别

    背景图片只能使用background-position调整位置

    插入图片可通过padding，margin调整位置，也可通过盒模型的padding及margin调整

  5. 清除标签默认的内外边距

    * {
    padding: 0;
    margin: 0
}
/*对于行内元素，一般只设置左右边距，不用设置上下边距，因为有些浏览器不支持行内元素的上下边距，如IE*/

  6. 外边距合并

    • 相邻块元素垂直外边距的合并

      当上下相邻的两个块元素相遇时，如果上面的元素由下外边距
      下面的元素有上外边距，则他们之间的垂直边距不是两个边距之和
      而是取两个值中的较大者，这种现象被称为相邻块元素垂直外边距的合并（外边距塌陷）

      解决：尽量只给一个盒子添加margin值

    • 嵌套块元素垂直外边距的合并（塌陷）

      对于两个嵌套关系的块元素，如果父元素没有上内边距以及边框
      父元素的上外边距与子元素的上外边距发生合并
      合并后的外边距为两者中的较大者

      解决：

      1. 为父元素添加上边框
      2. 为父元素指定上内边距
      3. 为父元素添加overflow：hidden

  7. 去掉列表默认样式

    list-style: none;

  8. 圆角边框

    border-radius: length;
/*圆：length可为50%
  矩形圆： length为高度一半*/

  9. 盒子阴影

    box-shadow: 水平阴影 | 垂直阴影 | ...
浮动：

  1. CSS布局机制

    • 普通流：块级元素独占一行，从上向下顺序排列
      行内元素按照顺序，从左到右顺序排列，遇到父元素边缘自动换行
    • 浮动：让盒子从普通流中浮起来，主要让多个块级盒子一行显示
    • 定位：将盒子定在浏览器的某一个位置

  2. 浮动

    float: none | left | right;
/*浮动元素之间没有缝隙，和真正的行内块有区别
  浮动元素只会影响后面的标准流，不会影响后面的标准流*/

  3. 清除浮动

    清除浮动主要为了解决父级元素因为子级浮动引起内部高度为0的问题。清除浮动之后，父元素会根据浮动的子盒子自动检测高度，父盒子有了高度就不会影响下面的标准流了。

    父元素 {
    clear: both | left | right;
    
}
/* left: 不允许左侧有浮动元素
   right: 
   both: 
*/

    • 额外标签法（W3C推荐）

      <div> </div>  /*在父元素内新加标签*/

.father {
    clear: both;    
    
}

    • 为父元素添加

      overflow: hidden | auto | scroll;

    • 使用after伪元素

      .clearfix:after: {
    content: "";
    display: block;
    height: 0;
    visibility: hidden;
    overflow: hidden;/*此句可缺省*/
    clear: both;
}
.clearfix {
    *zoom: 1;  /*ie6,ie7专门清除浮动的样式*/
}

    • 使用双伪元素清除浮动

      .clearfix:after:
.clearfix:before: {
    content: "";
    display: table;
}
.clearfix:after {
    clear: both;
}
.clearfix {
    *zoom: 1;  /*ie6,ie7专门清除浮动的样式*/
}

定位

定位：将盒子定在某一个位置 自由的漂浮在其他盒子上面

定位 = 定位模式 + 边偏移

  1. 边偏移

    left: 20px | % ; /*左侧偏移量，定义元素相对其父元素左边线的距离*/
bottom:
right:
top:

  2. 定位模式

    position: static | relative | absolute | fixed

    • static

      默认定位，无定位的意思，按照标准流特性摆放位置，无边偏移，几乎不同

    • relative

      相对定位是元素相对于原来位置进行移动

      特点：原来在标准流中的区域继续占用，后面的盒子仍然以标准流的方式对待它

    • absolute

      绝对定位

      若父元素无定位，则以浏览器为准

      若父元素有定位，则以父元素为准

      特点：不保留位置，原来区域不再占用

    • fixed

      完全脱标

      只以浏览器视图（屏幕）为准

      ie6等低版本不支持

    • 绝对定位盒子居中

      left: 50%;
margin-left: -100px; /*自身宽度一半*/

    • 堆叠顺序

      z-index: 正整数 | 负整数 | 0
/*默认为0，数值越大，盒子越靠上，数字后不能加单位*/
